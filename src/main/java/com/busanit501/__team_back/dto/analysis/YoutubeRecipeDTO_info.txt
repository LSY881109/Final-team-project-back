# YoutubeRecipeDTO 클래스 설명

`YoutubeRecipeDTO`는 YouTube Data API를 통해 조회한 레시피 영상의 정보를 담기 위한 DTO(Data Transfer Object)입니다.

서비스 계층(`YoutubeApiService`)에서 생성되어 컨트롤러로 전달되며, 최종적으로 클라이언트에게 JSON 형태로 응답됩니다.

---

## 1. 필드 정보

-   `private String title;`
    -   **설명:** YouTube 영상의 제목을 저장합니다.
    -   **예시:** "초간단 김치찌개 만들기"

-   `private String thumbnailUrl;`
    -   **설명:** 영상의 썸네일 이미지 URL을 저장합니다. 클라이언트에서 미리보기 이미지로 사용됩니다.
    -   **예시:** "https://i.ytimg.com/vi/..."

-   `private String videoId;`
    -   **설명:** YouTube 영상의 고유 ID를 저장합니다. 이 ID를 사용하여 영상을 재생하거나 링크를 만들 수 있습니다.
    -   **예시:** "abcdef12345"

-   `private String description;`
    -   **설명:** 영상의 상세 설명을 저장합니다.
    -   **예시:** "누구나 10분 만에 만들 수 있는..."

---

## 2. JSON 응답 예시

클라이언트가 API를 통해 받게 될 `YoutubeRecipeDTO` 객체 하나의 JSON 표현 예시입니다.

```json
{
  "title": "초간단 김치찌개 만들기 (10분 완성)",
  "thumbnailUrl": "https://i.ytimg.com/vi/abcdef12345/hqdefault.jpg",
  "videoId": "abcdef12345",
  "description": "누구나 10분 만에 만들 수 있는 황금 레시피! #김치찌개 #요리"
}
```
